## 🚀 How to Use

### Option 1: Run the API Server
```bash
python3 run_server.py --reload
# Open http://localhost:8000/docs for Swagger UI
```

### Option 2: Programmatic Usage
```python
from main import ApplicationContext

ctx = ApplicationContext()
event = Event(id="evt_001", display_name="Summit", ...)
ctx.event_repo.create(event)
```

### Option 3: Embedded in Another App
```python
from main import create_app
import uvicorn

app = create_app()
uvicorn.run(app, host="0.0.0.0", port=8000)
```

## 🏗️ Architecture Highlights

### Event-Scoped Design (Phase B)
Projects are now scoped to events:
- Query: `GET /v1/events/{eventId}/projects`
- Model: `ProjectDefinition.event_id` (required)
- Storage: Hierarchical organization by event

### Knowledge Workflow
```
Extract → Draft Artifact → Review → Approve → Publish → Distribute
```

### Graph Alignment
All responses include Graph metadata:
- `@odata.type` - Entity type identifier
- `@odata.etag` - Concurrency token
- `@odata.context` - Metadata reference
- Proper error format with code/message
